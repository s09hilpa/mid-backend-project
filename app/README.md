# Frontend (Not Required)

This project is a **backend skeleton project**.

A frontend application is **not required** for this repository.

The `app/` folder exists only to respect the standard Hack Your Future project structure, where repositories are typically organized into:

- `api/` – Backend code
- `app/` – Frontend code

Even though this project does not require a frontend, the folder is included to remain consistent with the HYF fullstack project structure.

You can safely ignore this folder unless you decide to build a frontend later.
